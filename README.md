# Spotify Wrapped — Reality Check

This project analyzes my personal Spotify listening data using raw JSON exports
to explore how Spotify Wrapped compares to metrics derived directly from the data.

## Tools Used
- Excel / Power Query (data cleaning & transformation)
- Power BI (interactive dashboard & storytelling)

## Key Features
- Total listening time & trend analysis
- Top artists & songs by actual listening time
- Monthly listening behavior
- Heatmap of listening hours
- Spotify Wrapped vs Reality comparison
